# Website voor Advocaat Kamal NAZIR


Dit project bevat de basisbestanden voor de website van advocaat Kamal NAZIR. De website is ontworpen om zakelijk, helder en uitnodigend te zijn, met informatie over diensten, contactgegevens en certificaten.


## Bestanden


- **index.html** – De hoofdstructuur van de website.
- **style.css** – De stijlopmaak voor de website.
- **README.md** – Uitleg en richtlijnen voor gebruik of uitbreiding.


## Talen
De website bevat een eenvoudige taalindicatie (Nederlands, Engels, Perzisch). Indien gewenst kan dit uitgebreid worden naar een volledig meertalig systeem.
